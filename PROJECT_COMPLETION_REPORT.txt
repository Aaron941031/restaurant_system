📡 API 端點清單

【認證 API】

【用戶 API】
  GET    /api/user/profile           - 個人資料
  POST   /api/user/exclusion         - 添加排除
  GET    /api/user/exclusions        - 查看排除清單
  DELETE /api/user/exclusion/{id}    - 移除排除

【餐廳 API】
  GET    /api/restaurant/all         - 所有餐廳
  GET    /api/restaurant/{id}        - 單一餐廳
  POST   /api/restaurant/save        - 新增餐廳
  POST   /api/restaurant/rate        - 新增評分
  GET    /api/restaurant/{id}/ratings - 查看評論

【推薦 API】
  GET    /api/recommend/personal     - 個人推薦
  GET    /api/recommend/group/{id}   - 群組推薦

【歷史記錄 API】
  GET    /api/history/me             - 個人歷史
  POST   /api/history/save           - 保存記錄
  DELETE /api/history/{id}           - 刪除記錄

【揪團 API】
  POST   /api/groups/create          - 建立群組
  POST   /api/groups/join            - 加入群組
  GET    /api/groups/{id}            - 查看群組
  POST   /api/groups/{id}/end        - 結束群組

🎯 快速啟動步驟
════════════════════════════════════════════════════════════════════════════

【1️⃣ 環境檢查】
   java -version              (需要 21+)
   mysql --version            (需要 8.0+)
   mvn -version               (需要 3.6+)

【2️⃣ 數據庫設置】
   mysql -u root -p < database_init.sql

【3️⃣ 應用配置】
   編輯 src/main/resources/application.properties
   修改:
     spring.datasource.username=root
     spring.datasource.password=你的密碼

【4️⃣ 啟動應用】
   mvn clean spring-boot:run

【5️⃣ 測試 API】
   curl -X POST http://localhost:8080/api/auth/register \
     -H "Content-Type: application/json" \
     -d '{"name":"test","email":"test@example.com","password":"test123"}'
